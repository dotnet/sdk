# Setup

## Arcade Skills Plugin

This repository uses the [dotnet/arcade-skills](https://github.com/dotnet/arcade-skills) plugin for shared AI skills like CI analysis.

### Installation

The arcade-skills plugin must be installed to get access to shared skills like CI analysis.

1. Add the marketplace:
   ```
   /plugin marketplace add dotnet/arcade-skills
   ```
2. Install the plugin:
   ```
   /plugin install dotnet-dnceng@dotnet-arcade-skills
   ```
3. View available skills:
   ```
   /skills
   ```
